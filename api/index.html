Supreme Leader's Shopping API Documentation

Authentication

POST /api/auth/login.php

This api handles user logins and returns the user id and details if successfully verified.

Payload
  • username*
  • password*
Response Example
  • "name": "Jonathan Tan Jia Wei"
  • "phone": "91234567"
  • "user_id": "5fa40d29e57d4"
  • "email": "jonathan@mail.com"

POST /api/auth/signup.php

This api handles user signups.

Payload
  • email*
  • password*
  • name*
  • postal_code*
  • address*
  • city*
  • country*
  • phone_number*

Items

GET /api/items/all.php

This api returns the expiry and order status for all items

Payload
  • none
Response Example
  • "01351492" :
    • "current_orders" : 5
    • "total_required" : 10
    • "expiry" : "2020-12-25"

GET /api/items/show_one.php

This api returns the expiry and order status for one item

URL Params
  • item_id=01351492
Response Example
  • "current_orders" : 5
  • "total_required" : 10
  • "expiry" : "2020-12-25"

PUT /api/items/order.php

This api handles the ordering logic and accepts the user's cart details

Payload
  • "user_id": "01351492"
  • "item_list": (array)
    • "0" =>
    • "item_id": "01351491"
    • "quantity" "1"
    • "1" =>
    • "item_id": "01351492"
    • "quantity": "2"
    • ...

POST /api/user/add_address.php

This api will add a new addresses

Payload
  • user_id*
  • postal_code*
  • address*
  • city*
  • country*

POST /api/user/add_credit_card.php

This api will add a new credit card

Payload
  • user_id*
  • card_number*
  • cardholder_name*
  • expiry* (MM/YY)
  • cvv_number*

POST /api/user/edit.php

This api will edit the user's name or email address

Payload
  • user_id*
  • fullname*
  • email*

DELETE /api/user/delete_address.php

This api will delete a specified address

Payload
  • user_id*
  • postal_code*
  • city*

DELETE /api/user/delete_credit_card.php

This api will delete a specified credit card

Payload
  • user_id*
  • card_number*
